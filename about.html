About

Welcome to the LEGO 2K Drive Encyclopedia!

This fan website aims to compile as much information as possible about the LEGO 2k Drive vehicles, bricks (packs), and other in-game collectibles - an encyclopedia of sorts.

It is made by the players for the players, and not endorsed or related in any way to the LEGO and 2K companies. They retains all (copy)rights to the assets used here.

To share any feedback, appreciation or suggestion, you can use one of the following chanels:
discord thread - reddit thread - github issues

Release notes can be found and followed in the above reddit and discord.

The site is served live from github, where you can find the sources. Contributions are welcome!
